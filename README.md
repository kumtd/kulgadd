<!-- INSTALLATION -->
## Installation

### Prerequisites

- libwebsockets

### Build

1. `mkdir kulgadd`
2. `cd kulgadd`
3. `git clone git@github.com:kumtd/kulgadd.git source`
4. `mkdir build`
5. `cd build`
6. `cmake ../source`
7. `make install`
